#숫자야구게임
###구현할 기능 목록
1. 3자리 랜덤한 숫자를 생성하는 기능  
    * [예외] 중복되는 숫자가 있는 경우  
    
2. 숫자를 입력받는 기능  
    * [예외] 숫자 이외의 문자를 입력받는 경우
    * [예외] 3자리 이상의 숫자를 입력받는 경우  

3. 볼, 스트라이크를 판단하여 출력하는 기능  
    * 3스트라이크일 경우 게임의 종료여부를 결정해야함.  
4. 게임의 종료 여부를 입력받는 기능  
    * 1이면 1번부터 다시 시작  
    * 2이면 종료되었음을 알림
