# 미션 - 숫자 야구 게임

## 기능 목록
- [X] 게임 반복 실행
  - [X] 종료 여부 입력 받기
  - [X] 입력에 따라 반복/종료 결정
- [X] 게임 진행
  - [X] 상대방 숫자 3개 결정
  - [X] 공격 숫자 입력
  - [X] 공격 숫자와 상대방 숫자 비교
  - [X] 비교 결과에 따라 게임종료/결과출력
- [X] 예외 처리
  - [X] 게임 반복 입력 검증
    - [X] 입력이 1 혹은 2인지 검증
  - [X] 공격 숫자 입력 검증
    - [X] 입력이 0을 포함하고 있는지 검증
    - [X] 입력이 세 자리인지 검증
    - [X] 입력이 숫자만 포함하는지 검증
    - [X] 입력이 중복 숫자를 포함하는지 검증
